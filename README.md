# ansipansi

## Prepare for WSL
1. *activatewsl.ps1*
2. *installwsl2kernal.ps1*
3. reboot system

## Install Ubuntu 22.04 LTS
1. *installmyubuntu.ps1*  
```./installmyubuntu.ps1 <wslname> <wslvhdxpath> <username>```  

**e.g.** 
```
./installmyubuntu.ps1 bingobongo c:\wsldistros\bingobongo dulli
```
